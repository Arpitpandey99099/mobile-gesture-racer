# mobile-gesture-racer
🏎️ Mobile Gesture Racer (AI-Powered)
✨ Key Features
No Install Required: Runs directly in Chrome, Safari, or Edge via GitHub Pages.

AI Hand Tracking: Uses the front-facing camera to track hand landmarks in real-time.

Advanced Gesture Controls:

Steer: Move your index finger left or right.

Nitro Boost (Gold Car): Open your palm (5 fingers) to double your speed and points.

Emergency Brake: Make a fist (0 fingers) to slow down and navigate tight gaps.

Mobile Optimized: Specifically tuned for mobile processors with low-latency tracking.

Instant Restart: Tap the screen after a crash to jump back into the action.

🛠️ Technical Stack
Frontend: HTML5, CSS3

Logic: JavaScript (ES6)

AI Engine: Google MediaPipe Hands

Camera Interface: MediaPipe Camera Utils
